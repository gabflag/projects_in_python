# A respeito do programa que foi encaminhado:
  Sua função principal é receber um numero inteiro na entrada e retornar se ele se encontra ou não se encontra em uma lista de cartões de crédito.
  Logo de inicio, fiz o tratamento de erros para o número digitado pelo usuário. Obviamente, não foi utilizado uma lista de cartões de crédito real, 
  então criei uma função especicifca para criar um número de cartão e outra para adiciona esse numero criado em uma lista. E no fim, uma breve função
  que retorna um verdadeiro/falso (boolean) se o numero de cartão digitado se encontra na lista.
  
# A respeito do programa de testes:
  O programa de testes é básico e pode ser melhorado, porém achei interessante fazer algumas validações e ressaltar
  métodos que podem ser adotados para executar testes.
  
# Conceitos abordados:

  ===> Programa principal:
  - Foi projetado para ser orientado a objetos.
  - Laço for para formatar um int.
  - Criar um número randomico não iniciado com zero e com numero de digitos pré-definido, usando a biblioteca random e sua função randint.
  - Criar uma lista e adicionando elementos a ela, e posterior, compara se existe certo elemento nessa lista.
